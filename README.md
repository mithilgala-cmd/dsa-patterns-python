## 🗂️ Repository Structure

Each problem is organized by **pattern**, making it easier to recognize and apply concepts across different questions.

```
arrays/
    two_sum.py
tests/
    arrays/
        test_two_sum.py
```

* Each solution has a corresponding **pytest test file**
* Clean and scalable folder structure
* Easy to revise before interviews

---

## 🛠️ Testing Setup

This repository uses **pytest** for automated testing.

### 🔧 Setup

```bash
pip install -r requirements.txt
```

### ▶️ Run Tests

```bash
python -m pytest
```

💡 Use `template.py` to quickly start solving new problems.
